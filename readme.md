# Portfolio Site

このプロジェクトは、Web コーダー転職用のポートフォリオサイトです。デザインカンプを参考にしたコーディング練習を兼ねています。

## 概要

- **目的**: Web コーダー転職用ポートフォリオ
- **技術構成**: HTML5 / CSS3 / JavaScript
- **最終形態**: WordPress テーマ化予定

## 制作フロー

1. 静的サイト（HTML/CSS/JS）として実装
2. WordPress テーマへの移行
3. ポートフォリオサイトへの掲載

## フォルダ構成

```
portfolio-site/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
└── README.md
```

## 開発環境

- エディタ: VS Code
- ローカル環境: Local（WordPress 化時に使用予定）
- バージョン管理: Git
